Downloading content :

CyberScript is heavily Modular.
to add features showcased, you need to Download a Datapack via the Native CyberScript tab in V's 1st Apartment PC. Once downloaded, Go to the mod menu in the pause screen, then go to the CyberScript Datapack Manager and enable your downloaded packs. (Yes you NEED Native UI settings mod)
A Datapack is a folder that contains Organized scripts that are to be executed by CyberScript's Script Engine, via triggers.
It can be missions, dialogs, places and events...
To access Keystone :
  • Activate PC in V's FIRST Apartement. A Keystone window will be Natively in the PC as a tab.
You will find Keystone in PC Tabs "CyberScript"
‚Äã
Steps for Starters:
  • FIRST, go to the pause menu to access mod menu. Then go to CyberScript Settings and make sure the Custom UI Scroll slider is at max. Save the setting by Quicksaving.
  • If there is a whole CS Update for the framework, Press the refresh button twice and it will show a new button to Update the whole mod. You might get lucky and have the Update prompt the first time you open the PC tab.
  • If you had to update the Mod in game, You will see a popup that will ask you to reload CET and then a Save. For better Practice, reload CET, Quicksave, and then Quickload.
  • Once back into the CyberScript Keystone, Go to the Datapack tab. on the datapack screen, Find "Main" to make sure it Doesn't need to be updated. Update it if needed.
Here you will find :
  • Mod Update if needed (only visible if you are not up to date)
  • Datapack Menu, to manage datapacks (download, delete, update)
  • Corpo Wars Network, which will show you the latest Multiplayer Corpo Wars Information.
  • Stock Market, to buy and sell stock actions from famous Night City brands.
  • Item Market, to buy and sell items, usefull for housing !
  • Multi Info, shows your account Information.
  • CyberScript Changelog
