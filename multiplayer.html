Multiplayer

CyberScript Multiplayer Current State :

For now you can :
  • Build together
  • See other player as npc avatar
  • See other player driving car
  • Perform script to other players or servers
You can't :
  • Do a playthrought together like coop
  • Fight each other
  • See the same quests/enemies/....

CyberScript has an instance system to let the players meet together in, or create one for yourself and friends. if an instance is set as NSFW, the player who has enabled Nudity Censorship will not see it, same for NSFW Datapacks in Keystone, they will only show for people that disable Censorship.

If you join it, we don't take any responsibilities about what you will see inside

BUT You must respect the community rules. Check them on Discord.

Also, by going inside it, You certify that You are an adult and aware of the explicit content.

You can also report bad users in our discord.
Also there is a bad word filter for messaging, the message will not be received if it contains a bad word that we have in our list(over 2000+ words)

A player can own one and only one instance. You can setup several parameters like :
  • Instance Privacy (Open, Password required (only friends of you can see it) or secret (only friends of you can see it)
  • Instance NSFW tag, means in this instance NSFW Datapacks can be executed.
  • Instance Read-Only, if set to true, means only you can build in it. If set to false, means every user can build in the building area

Be Online

You will see other playerd and vehicles if they drive in a range of 30 around you.
Our server can handle 1000 users at the same time, but you will not see them. The game limitation about it is 50. (and I think it's enough ;))
Please understand that we are a community project and not professional so it can have bugs.
Now you are online, you can perform several actions :
If you are a player :
  • Manage and add friends
  • Make for a guild
  • Message friends, guilds and instances
  • whisper to a player
  • Shoot talk to players around you
If you are the owner of the instance :
  • Manage users (kick, ban, unban)
If you are the owner of a Guild:
  • Manage members (kick, ban, unban)

Online Datapacks

Multiplayer Coops

You can Heal someone by using a heal item. WIth this pack, Think of the things that can be made from this idea....

Multiplayer Builder

if you are the owner, you can define a custom multiplayer area for building. Use Create or delete custom place area to manage buildable areas.
For the others, This datapack works exactly like Places of Night City datapack execp that you can't buy or own the Multiplayer place (for now^^). It's just community building.

Multiplayer RolePlay Joytoy(NSFW)

You can be a joytoy or a client online.
Client : You can find a player that is a Joy Toy and ask for several "services" in exchange of money.
"Animation" (Fade out then in) and sound will be played for you and the Joy Toy.
Joy Toy : You will win money from players that ask you to perform your talents.
Choose your role in the interface by selecting Multiplayer RP Joytoy Datapack in interact UI.

Hide N Seek (WIP, not published before 15/03/2022)

One player needs to find other players in the city. If a game is started, you cannot join it. game stops when all players have been found OR when the Seeker stops the game OR when there is no hiders

Guild System (WIP)

Players can team up in Guilds.
Any player can create a Guild. Then other players can ask you to join it.
Guilds are related to gangs in the game and they fight against other Guilds.
On your map, if you are online, you will not see Base Game Gangs you will see Guild territories.
A Guild can Lose or Win a territory by making some events (WIP, not published before Fall 2022/2023)
