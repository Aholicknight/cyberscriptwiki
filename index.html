What is CyberScript :

Based on CET, this mod is a powerful Scritping Engine Framework for Cyberpunk 2077.

it allows you to make a lot of new things (more than infinite ♾) in the game.
It is both for creators and players.
Creators can make and share content for Players that will play it with this mod.
A Collection of scripts is named a datapack, you can think of it as a Mod within a Mod.
A default Datapack is provided for immediate play, and the mod also has a complete Scripting Framework to Create/Edit custom missions and events.

Main functions:

A highly flexible Scripting Engine with trigger, action system, conditional statement, logic operation and function definition to achieve the following functions：
Technical Content :

  • Editor in Game

  • Custom Quest, Dialog, Subtitle, Shard, Phone Conversation, Circuit with Path, Faction, Event, Fixer, Housing, Place, NPC, Sounds, Texture and Radio
  • A framework to design your own world in Night city !
The quest types in this mod are not hard-coded. You can freely combine triggers and actions to make your own quests and dialogues, which leads to infinite possibilities. We call that Scripting
